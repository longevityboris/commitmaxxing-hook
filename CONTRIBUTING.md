# Contributing

Want to make the hook even better? Great.

## How to contribute

1. Fork the repo
2. Create a branch (`git checkout -b my-improvement`)
3. Make your changes
4. Test that the hook still works with Claude Code
5. Commit with a descriptive message
6. Open a PR

## What we're looking for

- Bug fixes
- Better commit message heuristics
- Support for more AI coding tools
- Documentation improvements

## What we're not looking for

- Complexity. The hook is 5 lines. That's a feature, not a limitation.
- Dependencies. It's a bash script. Let's keep it that way.

## Code style

Keep it simple. If your change adds more than a few lines, think hard about whether it's necessary.
